O sistema foi desenvolvido em Java para desktop, com o banco de dados SQLite. O SQLite foi escolhido por ser leve e não necessitar de servidor ou qualquer tipo de instalação.

O banco é criado automáticamente (banco.db) caso não exista. Inicialmente, o banco vem populado com três usuários:
- Um administrador (usuário "admin" e senha "12345").
- Um operador (usuário "operador" e senha "12345").
- Um gestor (usuário "gestor" e senha "12345").

Serão desenvolvidos os casos de uso:
- Efetuar login
- Manter funcionários
- Manter lojas
